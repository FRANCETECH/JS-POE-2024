HTTP

Le protocole HTTP est la manière dont les navigateurs web et les serveurs communiquent entre eux.

Concept essentiels

Méthodes : GET : Récupèrer des données POST : Soumettre des données PUT: Met à jour une ressource spécifique avec des données fournies DELETE : Supprime une ressources spécifique Code de statut: 2xx : Succès (ex: 200 OK) 3xx : Rédirection (ex : 301 : Moved Permanently) 4xx : Erreurs coté client (ex 404 NOT FOUND) 5xx : Erreurs coté serveurs (ex 500 INTERNAL SERVER ERROR )
